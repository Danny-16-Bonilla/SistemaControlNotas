# Sistema de Gestión y Control de Notas Estudiantiles (SGCE)

Bienvenido al repositorio oficial del proyecto SGCE. Este sistema web está diseñado para optimizar y automatizar la gestión, control y consulta de calificaciones académicas en instituciones de nivel universitario.

---

## Estructura del Proyecto (Fase I) YA ENTREGADA

El repositorio está organizado de la siguiente manera para facilitar la colaboración del equipo:

- **documentos/**: Documentación textual del sistema.
    - requisitos/: Requisitos funcionales y no funcionales de la Fase I.
    - historias_usuario/: Historias de usuario para el desarrollo ágil.
    - arquitectura/: Definición de la estructura del sistema.
    - adr/: Registros de Decisión de Arquitectura (Architecture Decision Records).
- **diagramas/**: Modelado visual.
    - uml/: Diagramas de casos de uso, clases y secuencias.
    - c4/: Modelado C4 (Contexto, Contenedores, Componentes).
