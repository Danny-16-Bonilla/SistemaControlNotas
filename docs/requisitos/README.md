# Requisitos del Proyecto
 
## Sistema de Gestión y Control de Notas Estudiantiles (SGCE)
 
### Objetivo
 
Definir los requisitos funcionales y no funcionales del sistema para garantizar un desarrollo organizado y acorde con las necesidades de la institución.
 
---
 
## Requisitos Funcionales
 
| Código | Requisito |
|---------|-----------|
| RF-01 | El sistema permitirá registrar estudiantes. |
| RF-02 | El sistema permitirá registrar docentes. |
| RF-03 | El administrador podrá crear, editar y eliminar usuarios. |
| RF-04 | Los docentes podrán ingresar, modificar y consultar calificaciones. |
| RF-05 | Los estudiantes podrán consultar sus notas por curso. |
| RF-06 | El sistema calculará automáticamente el promedio de cada estudiante. |
| RF-07 | El sistema generará reportes de calificaciones. |
| RF-08 | El sistema permitirá buscar estudiantes por nombre o identificación. |
 
---
 
## Requisitos No Funcionales
 
| Código | Requisito |
|---------|-----------|
| RNF-01 | El sistema deberá ser fácil de utilizar. |
| RNF-02 | Solo usuarios autorizados podrán acceder mediante autenticación. |
| RNF-03 | La información se almacenará de forma segura. |
| RNF-04 | El sistema deberá ser compatible con los principales navegadores web. |
| RNF-05 | El tiempo de respuesta no deberá superar los 3 segundos en operaciones normales. |
| RNF-06 | La documentación se desarrollará utilizando Markdown y GitHub. |
