# Historias de Usuario
 
## Sistema de Gestión y Control de Notas Estudiantiles (SGCE)
 
### HU-01
 
**Como** administrador,
 
**quiero** registrar estudiantes,
 
**para** mantener actualizada la información académica.
---
 
### HU-02
 
**Como** docente,
 
**quiero** registrar y modificar las calificaciones,
 
**para** evaluar el rendimiento académico de los estudiantes.
 
---
 
### HU-03
 
**Como** estudiante,
 
**quiero** consultar mis calificaciones,
 
**para** conocer mi desempeño académico.
 
---
 
### HU-04
 
**Como** administrador,
 
**quiero** administrar usuarios,
 
**para** controlar el acceso al sistema.
 
---
 
### HU-05
 
**Como** docente,
 
**quiero** generar reportes de notas,
 
**para** facilitar la entrega de resultados académicos.
 
---
 
## Criterios de aceptación
 
- El usuario debe iniciar sesión para acceder al sistema.
- Solo los docentes podrán registrar y modificar calificaciones.
- Los estudiantes únicamente podrán consultar sus propias notas.
- El administrador tendrá acceso a todas las funciones del sistema.
